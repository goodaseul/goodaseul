**배열함수

 - forEach() => {반환값 없음 / 변수 필요없음}
/return 있음
 - map() => {반환값 있음 return / 반드시 배열을 반환} //객체 리턴할 때 많이 사용
 - filter() => {조건에 대한(true/false)값으로 참인 값만 값을 배열을 반환 return )
 - some() => {조건에 해당하는 거가 하나라도 있다면 true/false}
 - every() => {모든 거가 조건에 해당하는지 true/false}
 - find() => {조건에 해당하는 거 값을 찾아줘 첫번째것만 return}
 - findIndex() => {조건에 해당하는 거 인덱스를 찾아줘 return}


item.startsWith("L") ==L 로 시작하는 아이템 찾아줘
