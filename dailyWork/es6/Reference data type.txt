자 object / array를 복사하는건 변수를 복사하는 거가아니란다

예시로 알려줄게~  ^^
let object = {name: 'park'};
let object2 = array;

이렇게 되면  object 2 에 name: 'park'이 나오겟찌?
근데 object.name = 'kim' 으로 바꾸면 object2 의 이름을 바꾼 적이 없는데 바껴있어
왜그런지 아니? 

object / object2 ===== > name:'park'이라는 저장소에서 가져와서 쓰고있는거란다?
그래서 저장소를 바꾸게 됨으로 할당되어있던 변수들이 다 바뀌는거란다



function 할당바꾸기(object){
	object.name = 'kim'
}

할당바꾸기(object);

할당바꾸기 (var object = {name : kim}) 이란 소리임
그러니가 파라미터가 일종의 변수처럼 생성되고 사라지는 존재임
새로운 변수에 새로운 화살표를 재할당한 것임 그래서 실제 기존에 있던 object를 건드리지않음
저장소가 탄생한거임. 


var 이름1 = { name : '김' };

function 변경(obj){
  obj.name  = 'park';
